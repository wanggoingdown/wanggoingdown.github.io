equals
今天来讲一讲这两天我看的Java里面比较常用的方法equals，don`t BB ,show the code

  public boolean equals(Object obj) {
    return (this == obj);
}

这是来自Object类的equals函数，只要对比一下形参与当前对象的地址是否一致就完事儿啦，那让我们再看看注释
`*

  • It is reflexive: for any non-null reference value

    • *     {@code x}, {@code x.equals(x)} should return
*     {@code true}.
* <li>It is <i>symmetric</i>: for any non-null reference values
*     {@code x} and {@code y}, {@code x.equals(y)}
*     should return {@code true} if and only if
*     {@code y.equals(x)} returns {@code true}.

？？？什么鬼，手动尼克杨，这可能是假注释，这是生怕我看不懂吗？x和y比较，如果y.equals(x)为true，那么x.equals(y)也为true。好吧说了等于没说~。~

初步总结八个基本数据类型boolean,byte,short,char,int,float,long,double是没有equals函数的，只能用==判断，类对象只要没有重写equals方法，那么默认equals（）跟==是等价的，都是对比内存地址

    包装类的equals
    先看看String.equals()的源码

    public boolean equals(Object anObject) {
    if (this == anObject) {
        return true;
    }
    if (anObject instanceof String) {
        String anotherString = (String) anObject;
        int n = length();
        if (n == anotherString.length()) {
            int i = 0;
            while (n-- != 0) {
                if (charAt(i) != anotherString.charAt(i))
                        return false;
                i++;
            }
            return true;
        }
    }
    return false;
}

    根据我多年的看代码经验，这代码没毛病（括弧笑），有人会说这段代码写得过于小心了，直接前三行判断地址就完事儿了，为什么非同一内存地址还要判断每一个字符呢？难道”123”不等于”123”?没错还真不等于，let me show you

     String a="123";
String b=new String("123");

    虽然a和b都为”123”但是b是new出来的，重新申请了内存地址所以地址就不一样了，所以后面的代码是为了防止这种情况发生，就强行比对了每一个字符。

    Integer的equals

    public boolean equals(Object obj) {
    if (obj instanceof Integer) {
        return value == ((Integer)obj).intValue();
    }
    return false;
}

    Double的equals

    public boolean equals(Object obj) {
    return (obj instanceof Double)
           && (doubleToLongBits(((Double)obj).value) ==
                  doubleToLongBits(value));
}

    Float的equals

    public boolean equals(Object obj) {
    return (obj instanceof Float)
           && (floatToIntBits(((Float)obj).value) == floatToIntBits(value));
}

    Boolean的equals

    public boolean equals(Object obj) {
    if (obj instanceof Boolean) {
        return value == ((Boolean)obj).booleanValue();
    }
    return false;
}

    Character的equals

    public boolean equals(Object obj) {
    if (obj instanceof Character) {
        return value == ((Character)obj).charValue();
    }
    return false;
}

    Long的equals

     public boolean equals(Object obj) {
    if (obj instanceof Long) {
        return value == ((Long)obj).longValue();
    }
    return false;
}

    Short的equals

     public boolean equals(Object obj) {
    if (obj instanceof Short) {
        return value == ((Short)obj).shortValue();
    }
    return false;
}
